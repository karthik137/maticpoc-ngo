# maticpoc-ngo
Matic poc ngo server


#### How to start the server?

###### Clone the repository
```
$ git clone https://github.com/karthik137/maticpoc-ngo.git

```

###### Install dependencies

```
$ cd maticpoc-ngo
$ npm install

```

###### Start the server

> NOTE: The ngo server runs at port 4000
```
$ npm start

```
